# stock-trading-simulator
Create a virtual environment with your usual procedure, referencing requirements.txt.
Start with inputting your initial portfolio and stock trader in config.py, then run combined_alg.py.
combined_alg.py runs a three-way split of your initial assets into the EMA+ARIMA strategy (ema_arima_only.py), Ichimoku Cloud strategy (ichimoku_cloud.py), and Mainwave strategy (mainwave_only.py).
